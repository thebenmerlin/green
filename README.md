# Green Eventwaala

Eco-friendly event management website built with Next.js 15, Sanity CMS, and Formspree.

## Quick Start

1. Clone this repository
2. Run: `npm install`
3. Add your Sanity Project ID to `lib/sanityClient.js`
4. Add your Formspree Form ID to `app/contact/page.js`
5. Run: `npm run dev`
6. Open http://localhost:3000

## Tech Stack
- Next.js 15 (App Router)
- React 19
- Tailwind CSS
- Sanity CMS
- Formspree (forms)

## Deployment
Deploy to Vercel for instant hosting.
